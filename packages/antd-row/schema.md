# Component Props API:

**_Properties_**

 - <b id="/properties/align">align</b>
	 - Type: `string`
	 - The value is restricted to the following: 
		 1. _"bottom"_
		 2. _"middle"_
		 3. _"stretch"_
		 4. _"top"_
 - <b id="/properties/justify">justify</b>
	 - Type: `string`
	 - The value is restricted to the following: 
		 1. _"center"_
		 2. _"end"_
		 3. _"space-around"_
		 4. _"space-between"_
		 5. _"start"_
 - <b id="/properties/attrs">attrs</b>
	 - Type: `object`
